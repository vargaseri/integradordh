Batalla Animusical
¿Como registrarme?

Ingresa desde Home, hace clic en registrate, completá tus datos (nombre completo, email, nombre de usuario, contraseña y pais), luego vas a estar listx para comenzar a jugar!

¿Como comienzo a jugar?

Desde el Home, en Comenzar -> Ingresá, logueate con tu usuario y password, luego seleccioná el personaje para la pelea. Hace clic en comenzar! En cada turno que tengas para atacar, se desplegará una letra de canción, y una lista de opciones con la continuación de la letra correcta a seleccionar, acorde al personaje elegido. Cuantas mas opciones sean las correctas, nuestro luchador Animusical arrojará mas golpes, para derrotar a tu rival!

Olvidé mi usuario o contraseña

Desde el Home, en Comenzar -> Ingresá, hacé clic en "¿olvidaste tus datos?", por último vas a completar con tu mail para restablecer el usuario o contraseña.

¿Como puedo ver mis datos de usuario?

Luego de loguearte, ingresá a Perfil -> Datos de Usuario y ahí podes ver toda tu información.

¿Como modificar mi foto de perfil?

Luego de loguearte, ingresá a Perfil -> Datos de Usuario -> Cambiar foto. Una vez seleccionada tu imagen de avatar, hacá clic en Confirmar cambios.

¿Como editar mis datos de perfil?

Luego de loguearte, ingresá a Perfil -> Datos de Usuario -> Modificar datos -> Confirmar cambios.

¿Como veo mi puntaje?

Luego de loguearte, ingresá a Perfil -> Datos de Usuario, allí podras ver la puntuación mas alta que lograste hasta ahora.

¿Quienes pueden jugar Batalla Animusical?

¡Batalla Animusical es un juego para los fanaticos de la música, divertite y demostrá tus conocimientos en cada pelea!
